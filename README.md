# PolyWatch

**PolyWatch** is a market integrity monitor for Polymarket, designed to detect anomalies, manipulation (Pump & Dump), and unusual volatility events using empirical data analysis.

## Project Goal
To build a system that monitors prediction markets for signs of manipulation, providing transparency and security insights for the decentralized ecosystem.
